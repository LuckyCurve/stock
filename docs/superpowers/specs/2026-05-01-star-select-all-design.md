# ⭐ 星标全选按钮

## 概述

在左侧面板添加一键勾选/取消所有星标股票的 toggle 按钮，方便快速对比收藏的股票。

## 位置与外观

- 位于搜索框下方、ticker 列表上方，独立一行
- 无星标股票时完全隐藏（不占空间）
- 按钮样式与面板风格一致，紧凑小巧

## 文案与行为

- **未全选时**：显示 `⭐ 勾选全部星标 (N)`，点击后勾选所有星标 ticker
- **已全选时**：显示 `⭐ 取消全部星标 (N)`，点击后取消所有星标 ticker 的勾选
- 判断逻辑：所有星标 ticker 都已勾选 → 显示"取消"，否则显示"勾选"
- N 为当前星标总数

## 勾选行为细节

- **勾选全部星标**：先清除当前所有勾选，再勾选所有星标 ticker（干净的"看我的收藏"体验）
- **取消全部星标**：仅取消星标 ticker 的勾选，不影响非星标 ticker 的勾选状态

## 超限处理

- 星标数 > 8 时，点击弹 alert 提示"最多同时对比 8 支股票"，不做任何勾选变更

## 交互反馈

- 点击后立即刷新左侧列表和右侧内容（与现有 toggleCheck 逻辑一致）
- 切换到"图表对比"tab 可直接看到结果

## 实现要点

- 修改文件：`src/template.html`
- 在 `renderTickerList()` 中，列表渲染前判断星标数，插入按钮 DOM
- 按钮点击处理函数 `toggleAllStars()`
- 每次 `renderTickerList()` 重新渲染时重建按钮，确保文案与状态同步
